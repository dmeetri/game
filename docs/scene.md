# Сцены

Полное руководство по работе со сценами

В движке существует поддержка работы сразу нескольких сцен. Код сцены находится в `core.js`. Так же код можно посмотреть ниже, в разделе [Код](#код)

## Создать сцену

> [!WARNING]
> При создании сцены важно понимать, что по умолчанию сцена имеет размер `1x1`, то есть 1 на 1 символ. Сцена не растягивается на всю свободную ширину экрана сама
